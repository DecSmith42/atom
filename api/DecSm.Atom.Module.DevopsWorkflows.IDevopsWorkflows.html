Remarks

Implementing this interface configures the necessary services for generating Azure DevOps Pipeline YAML files, providing Azure DevOps-specific workflow variables, and adapting build paths and reporting when running within Azure DevOps Pipelines.

Remarks

This method registers DecSm.Atom.Module.DevopsWorkflows.Workflows.Devops.DevopsWorkflowFileWriter for generating workflow files and DecSm.Atom.Module.DevopsWorkflows.DevopsVariableProvider for Azure DevOps-specific workflow variables. When running inside Azure DevOps Pipelines, it also sets up DecSm.Atom.Module.DevopsWorkflows.DevopsSummaryOutcomeReportWriter for reporting and adjusts artifact and publish paths to Azure DevOps conventions.

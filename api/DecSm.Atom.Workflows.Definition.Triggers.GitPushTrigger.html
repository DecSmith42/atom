Class GitPushTrigger

Namespace
DecSm.Atom.Workflows.Definition.Triggers
Assembly
DecSm.Atom.Workflows.dll

Represents a workflow trigger that activates when code is pushed to a Git repository.

[PublicAPI]
public sealed record GitPushTrigger : IWorkflowTrigger
Inheritance
object
GitPushTrigger
Implements
IWorkflowTrigger

Examples

// Trigger on pushes to the main branch only
var mainTrigger = GitPushTrigger.ToMain;
// Trigger on pushes to any branch, excluding documentation changes
var featureTrigger = new GitPushTrigger
{
    ExcludedPaths = ["docs/**", "*.md"]
};

Remarks

This trigger supports filtering by branches, file paths, and tags to control workflow execution. When filter lists are empty, no filtering is applied for that category.

Remarks

This interface defines the parameters required to connect to an Azure Key Vault and configures the necessary services for retrieving secrets.

Properties

AzureKeyVaultValueInjections

Configures how Azure Key Vault parameter values are injected into the build process.

AzureKeyVaultValueInjections AzureKeyVaultValueInjections { get; }

Property Value

AzureKeyVaultValueInjections

Remarks

This property allows customization of the source for each Azure Key Vault related parameter. By default, most parameters are expected from environment variables, while the app secret is expected from a secure secret store.

AzureVaultAddress

Gets the address (URI) of the Azure Key Vault.

[ParamDefinition("azure-vault-address", "Address for the Azure Vault", ParamSource.All, null)]
string? AzureVaultAddress { get; }

Property Value

string

Remarks

This parameter is crucial for identifying which Azure Key Vault instance to connect to. It can be provided via command-line arguments, environment variables, or appsettings.json.

AzureVaultAppId

Gets the Application (Client) ID for the App Registration that has access to the Azure Key Vault.

[ParamDefinition("azure-vault-app-id", "Azure ID for App Registration that has access to the Azure Vault", ParamSource.All, null)]
string? AzureVaultAppId { get; }

Property Value

string

Remarks

This ID is used for service principal authentication to Azure Key Vault. Ensure the registered application has appropriate permissions (e.g., Get, List) on secrets in the vault.

AzureVaultAppSecret

Gets the client secret for the App Registration that has access to the Azure Key Vault.

[ParamDefinition("azure-vault-app-secret", "Azure Secret for App Registration that has access to the Azure Vault", ParamSource.All, null)]
string? AzureVaultAppSecret { get; }

Property Value

string

Remarks

This secret is used in conjunction with the AzureVaultAppId for authenticating the application to Azure Key Vault. It should be treated as sensitive information.

AzureVaultAuthPort

Gets the port number for the local authentication redirect during the Azure Key Vault authentication process.

[ParamDefinition("azure-vault-auth-port", "Port for local auth redirect (default 3421). Set to 0 to use a random port.", ParamSource.All, null)]
int AzureVaultAuthPort { get; }

Property Value

int

Remarks

This port is used when performing device code or interactive authentication flows. The default port is 3421. Setting this to 0 will select a random available port.

AzureVaultTenantId

Gets the Tenant ID associated with the Azure Key Vault.

[ParamDefinition("azure-vault-tenant-id", "Tenant ID for the Azure Vault", ParamSource.All, null)]
string? AzureVaultTenantId { get; }

Property Value

string

Remarks

The Tenant ID is part of the authentication process to ensure access to the correct Azure Active Directory tenant.

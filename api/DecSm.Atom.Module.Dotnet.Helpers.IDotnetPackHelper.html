DotnetPackAndStage(RootedPath, DotnetPackAndStageOptions?, CancellationToken)

Packs a .NET project specified by its path and stages the resulting NuGet package.

[PublicAPI]
Task DotnetPackAndStage(RootedPath projectPath, DotnetPackAndStageOptions? options = null, CancellationToken cancellationToken = default)

Parameters

projectPath RootedPath

The full path to the project file (e.g., "src/MyProject/MyProject.csproj").

options DotnetPackAndStageOptions

Optional. Configuration options for the packing and staging process.

cancellationToken CancellationToken

A cancellation token to observe while waiting for the task to complete.

Returns

Task

A System.Threading.Tasks.Task representing the asynchronous operation.

Remarks

This method performs the following steps: 

<ol><li>Determines the output directory for the NuGet package.</li><li>Cleans up existing package directories if specified.</li><li>
            Applies version transformations to project files if
            <xref href="DecSm.Atom.Module.Dotnet.Helpers.DotnetPackAndStageOptions.SetVersionsFromProviders" data-throw-if-not-resolved="false"></xref> is <code>true</code>.
        </li><li>
            Executes the `dotnet pack` command using
            <xref href="DecSm.Atom.Module.Dotnet.Cli.IDotnetCli.Pack(DecSm.Atom.FileSystem.RootedPath%2cDecSm.Atom.Module.Dotnet.Cli.PackOptions%2cDecSm.Atom.Process.ProcessRunOptions%2cSystem.Threading.CancellationToken)" data-throw-if-not-resolved="false"></xref>.
        </li><li>Locates the generated `.nupkg` file.</li><li>Moves the `.nupkg` file to the Atom publish directory for the project.</li></ol>

The BuildVersion is used for naming the NuGet package and for version transformation.

Exceptions

StepFailedException

Thrown if the packed NuGet package is not found after the operation.

DotnetPackAndStage(string, DotnetPackAndStageOptions?, CancellationToken)

Packs a .NET project by its name and stages the resulting NuGet package.

[PublicAPI]
Task DotnetPackAndStage(string projectName, DotnetPackAndStageOptions? options = null, CancellationToken cancellationToken = default)

Parameters

projectName string

The name of the project to pack (e.g., "MyProject.Core").

options DotnetPackAndStageOptions

Optional. Configuration options for the packing and staging process.

cancellationToken CancellationToken

A cancellation token to observe while waiting for the task to complete.

Returns

Task

A System.Threading.Tasks.Task representing the asynchronous operation.

Remarks

This method first locates the project file based on its name, then calls DotnetPackAndStage(RootedPath, DotnetPackAndStageOptions?, CancellationToken) with the resolved path.

Exceptions

StepFailedException

Thrown if the project file cannot be located.

AddAtom<TBuilder, TBuild>(TBuilder, string[])

Configures the Microsoft.Extensions.Hosting.IHostApplicationBuilder with all necessary Atom services, dependencies, logging, and default settings.

public static TBuilder AddAtom<TBuilder, TBuild>(this TBuilder builder, string[] args) where TBuilder : IHostApplicationBuilder where TBuild : BuildDefinition

The host application builder instance to configure.

Returns

Type Parameters

TBuilder

The type of the host application builder.

TBuild

The type of the build definition, which must implement BuildDefinition.

Remarks

This method performs the following key configurations:

  • Registers the DecSm.Atom.Build.AtomService as a hosted service.
  • Registers the build definition (TBuild) and related interfaces.
  • Adds core Atom services like IParamService, ReportService, and Spectre.Console.IAnsiConsole.
  • Registers default providers for build information: IBuildIdProvider, IBuildVersionProvider, and IBuildTimestampProvider.
  • Configures file system access via IAtomFileSystem.
  • Registers build execution and workflow generation services.
  • Sets up logging with Spectre.Console and report providers, filtering out verbose Microsoft host logs.
  • Parses command-line arguments and makes them available as CommandLineArgs.
  • Resolves and registers the BuildModel.
  • Invokes ConfigureBuildHostBuilder(IHostApplicationBuilder) if implemented by the build definition.

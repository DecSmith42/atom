Represents the complete model of a build, including all targets and their states.

[PublicAPI]
public sealed record BuildModel
Inheritance
object
BuildModel

Remarks

This model is generated by Atom on startup and serves as the primary source of information for the build. It is immutable by design, except the values in TargetStates.

InstallTool(string, string?, bool, bool)

Installs or updates a .NET CLI tool.

[PublicAPI]
void InstallTool(string toolName, string? version = null, bool global = true, bool forceReinstall = false)

Parameters

toolName string

The name of the tool to install (e.g., "dotnet-format").

version string

Optional. The specific version of the tool to install. If not specified, the latest stable version is used.

global bool

If true, the tool is installed globally. If false, it's installed locally to the current directory.

forceReinstall bool

If true, the tool will be reinstalled even if already present.

Remarks

This method first checks if the tool is already installed (unless forceReinstall is true or running in headless mode). If not installed or reinstallation is forced, it executes `dotnet tool update`.

If installing locally (global is false) and a tool manifest (`.config/dotnet-tools.json`) does not exist, it will be created.

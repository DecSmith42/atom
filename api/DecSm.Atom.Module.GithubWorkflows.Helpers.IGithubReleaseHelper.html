Methods

UploadArtifactToRelease(string, string, bool)

Uploads a build artifact to a specified GitHub Release.

[PublicAPI]
Task UploadArtifactToRelease(string artifactName, string releaseTag, bool dryRunWhenNotRunningInGithubActions = true)

Parameters

artifactName string

The name of the artifact to upload. This name should correspond to a directory within the Atom artifacts directory.

releaseTag string

The tag of the GitHub Release to which the artifact should be uploaded (e.g., "v1.0.0").

dryRunWhenNotRunningInGithubActions bool

If true (default), the upload operation will be simulated (logged but not executed) when the build is not running in a GitHub Actions environment.

Returns

Task

A System.Threading.Tasks.Task representing the asynchronous operation.

Remarks

The artifact will be zipped before uploading if it's a directory. The method will attempt to find an existing release with the given tag.

UploadAssetToRelease(string, RootedPath, bool)

Uploads a generic asset (file or directory) to a specified GitHub Release.

[PublicAPI]
Task UploadAssetToRelease(string releaseTag, RootedPath assetPath, bool dryRunWhenNotRunningInGithubActions = true)

Parameters

releaseTag string

The tag of the GitHub Release to which the asset should be uploaded (e.g., "v1.0.0").

assetPath RootedPath

The path to the asset (file or directory) to upload.

dryRunWhenNotRunningInGithubActions bool

If true (default), the upload operation will be simulated (logged but not executed) when the build is not running in a GitHub Actions environment.

Returns

Task

A System.Threading.Tasks.Task representing the asynchronous operation.

Remarks

If assetPath points to a directory, it will be zipped into a .zip file with the same name before uploading. If it points to a file that is not a .zip file, it will also be zipped.

Exceptions

InvalidOperationException

Thrown if the GitHub repository ID cannot be parsed.

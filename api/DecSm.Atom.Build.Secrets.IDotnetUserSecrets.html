An interface that, when implemented on a build definition, enables sourcing secrets from the .NET user secrets store.

[PublicAPI]
public interface IDotnetUserSecrets

Examples

// 1. Implement the interface in your build definition
[BuildDefinition]
partial class Build : IDotnetUserSecrets, IMyTarget;
// 2. Define a secret parameter
[SecretDefinition("my-secret", "A secret from user secrets.")]
string MySecret => GetParam(() => MySecret);
// The value for "my-secret" will now be resolved from the user secrets store.

Remarks

This interface uses source generation to automatically register the DecSm.Atom.Build.Secrets.DotnetUserSecretsProvider with the dependency injection container. This allows the IParamService to resolve secrets marked with SecretDefinitionAttribute from the local user secrets store, which is ideal for development-time secrets that should not be committed to source control.

This interface is automatically included in the default BuildDefinition.

Provides base functionality for accessing services and parameters within the Atom build system.

[PublicAPI]
public interface IBuildAccessor

Remarks

This interface serves as a foundation for target definitions and helpers, offering convenient access to logging, file system operations, process execution, and parameter retrieval.

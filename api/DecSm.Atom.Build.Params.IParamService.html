Defines a service for resolving and managing build parameters from various sources.

[PublicAPI]
public interface IParamService

Remarks

The service resolves parameters from the following sources in order of precedence:

  1. Cache (if enabled)
  2. Command-line arguments
  3. Environment variables
  4. Configuration files (e.g., appsettings.json)
  5. Secret providers
  6. Interactive console prompts
  7. Default value

This service is central to the framework's parameter handling, providing features for type conversion, secret masking, and caching.

Remarks

These extensions exist so that CreateAsync and one or more AddAsync calls can be written as a single expression without intermediate variables:

await using var scope = await TransformFileScope
    .CreateAsync(file, _ => "v1")
    .AddAsync(c => c + "-patched");

Without them, the caller would need to await CreateAsync into a local and then await AddAsync separately.

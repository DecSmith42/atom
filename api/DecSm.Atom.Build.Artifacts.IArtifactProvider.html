Defines a provider for storing and retrieving build artifacts.

[PublicAPI]
public interface IArtifactProvider

Remarks

Implementations of this interface abstract the underlying storage mechanism, allowing for different backends such as cloud storage or local file systems. The Atom framework uses these providers to manage artifacts for targets like IStoreArtifact and IRetrieveArtifact.

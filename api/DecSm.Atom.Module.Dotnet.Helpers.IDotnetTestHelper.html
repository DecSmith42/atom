DotnetTestAndStage(RootedPath, DotnetTestAndStageOptions?, CancellationToken)

Runs tests for a .NET project specified by its path, then stages the test results and optionally code coverage reports.

[PublicAPI]
Task<int> DotnetTestAndStage(RootedPath projectPath, DotnetTestAndStageOptions? options = null, CancellationToken cancellationToken = default)

Parameters

projectPath RootedPath

The full path to the project file (e.g., "tests/MyProject.Tests/MyProject.Tests.csproj").

options DotnetTestAndStageOptions

Optional. Configuration options for the testing and staging process.

cancellationToken CancellationToken

A cancellation token to observe while waiting for the task to complete.

Returns

Task<int>

A System.Threading.Tasks.Task<TResult> that returns the exit code of the dotnet test command.

Remarks

This method performs the following steps: 

<ol><li>
            Configures `dotnet test` options, including TRX and HTML loggers, and optionally code
            coverage.
        </li><li>Cleans up existing test result directories.</li><li>
            Applies version transformations to project files if
            <xref href="DecSm.Atom.Module.Dotnet.Helpers.DotnetTestAndStageOptions.SetVersionsFromProviders" data-throw-if-not-resolved="false"></xref> is <code>true</code>.
        </li><li>
            Executes the `dotnet test` command using
            <xref href="DecSm.Atom.Module.Dotnet.Cli.IDotnetCli.Test(DecSm.Atom.FileSystem.RootedPath%2cDecSm.Atom.Module.Dotnet.Cli.TestOptions%2cDecSm.Atom.Process.ProcessRunOptions%2cSystem.Threading.CancellationToken)" data-throw-if-not-resolved="false"></xref>.
        </li><li>Copies the generated HTML test report to the Atom publish directory.</li><li>Generates a detailed test report using <xref href="DecSm.Atom.Module.Dotnet.Helpers.IDotnetTestHelper.GenerateTestReport(System.String%2cSystem.String%2cSystem.String%2cSystem.String%2cSystem.Boolean)" data-throw-if-not-resolved="false"></xref>.</li><li>
            If <xref href="DecSm.Atom.Module.Dotnet.Helpers.DotnetTestAndStageOptions.IncludeCoverage" data-throw-if-not-resolved="false"></xref> is <code>true</code>, installs
            `dotnet-reportgenerator-globaltool` and generates code coverage reports.
        </li><li>Generates a code coverage report using <xref href="DecSm.Atom.Module.Dotnet.Helpers.IDotnetTestHelper.GenerateCoverageReport(System.String%2cSystem.String%2cSystem.String%2cSystem.String%2cSystem.Boolean)" data-throw-if-not-resolved="false"></xref>.</li></ol>

The BuildVersion is used for version transformation.

DotnetTestAndStage(string, DotnetTestAndStageOptions?, CancellationToken)

Runs tests for a .NET project by its name, then stages the test results and optionally code coverage reports.

[PublicAPI]
Task<int> DotnetTestAndStage(string projectName, DotnetTestAndStageOptions? options = null, CancellationToken cancellationToken = default)

Parameters

projectName string

The name of the project to test (e.g., "MyProject.Tests").

options DotnetTestAndStageOptions

Optional. Configuration options for the testing and staging process.

cancellationToken CancellationToken

A cancellation token to observe while waiting for the task to complete.

Returns

Task<int>

A System.Threading.Tasks.Task<TResult> that returns the exit code of the dotnet test command.

Remarks

This method first locates the project file based on its name, then calls DotnetTestAndStage(RootedPath, DotnetTestAndStageOptions?, CancellationToken) with the resolved path.

Exceptions

StepFailedException

Thrown if the project file cannot be located.

The directory component of the path if the path resolves to an existing directory; otherwise null. Returns null (rather than throwing) when the path points to a file or does not exist.

Remarks

This delegates to System.IO.Abstractions.IPath.GetDirectoryName(string), which strips the last path segment and trailing separator — it does not return the directory's own name.

A new RootedPath representing the parent directory, or null if the current path is already a root.

Remarks

A trailing directory separator on the input path is stripped before the parent is computed so that /foo/ and /foo both yield the same parent (/). The returned path always ends with System.IO.Abstractions.IPath.DirectorySeparatorChar so it can be combined further with the / operator without producing double separators.

Remarks

Using an operator instead of a method keeps path-construction code concise and readable, e.g. atomFileSystem.GetPath("Root") / "src" / "MyProject". The operator preserves the original FileSystem reference so the resulting path remains bound to the same file-system abstraction.

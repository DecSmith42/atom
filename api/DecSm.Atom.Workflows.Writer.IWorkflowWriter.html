Defines a contract for writers that can generate and validate platform-specific workflow files.

[PublicAPI]
public interface IWorkflowWriter

Remarks

This interface is part of a strategy pattern where different writers handle different CI/CD platforms. Implementations are typically registered as singletons and are expected to be thread-safe.

Remarks

The primary purpose of IValidateBuild is to enhance the reliability and maintainability of Atom build scripts by catching common configuration mistakes early in the process.

Properties

ValidateBuild

Gets the build target that performs validation checks on the Atom build.

Target

Examples

This target is included in BuildDefinition and is automatically available if the build inherits from that type. Otherwise, it can be manually included:

public class Build : BuildDefinition
{
    // ... other targets ...
    // The ValidateBuild target from IValidateBuild would be automatically available
    // and could be set as a dependency for other targets.
    Target MyCustomTarget => t => t
        .DependsOn(nameof(IValidateBuild.ValidateBuild)) // ValidateBuild is the property from IValidateBuild
        .Executes(() => { /* Do something after validation */ });
}

The target can then be run with

atom ValidateBuild

Exceptions

StepFailedException

Thrown if the validation process encounters critical errors that prevent the build from proceeding. (Currently, this condition is not met as the 'errors' list is not populated).

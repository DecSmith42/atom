DotnetPublishAndStage(RootedPath, DotnetPublishAndStageOptions?, CancellationToken)

Publishes a .NET project specified by its path and stages the output.

[PublicAPI]
Task DotnetPublishAndStage(RootedPath projectPath, DotnetPublishAndStageOptions? options = null, CancellationToken cancellationToken = default)

Parameters

projectPath RootedPath

The full path to the project file (e.g., "src/MyWebApp/MyWebApp.csproj").

options DotnetPublishAndStageOptions

Optional. Configuration options for the publishing and staging process.

cancellationToken CancellationToken

A cancellation token to observe while waiting for the task to complete.

Returns

Task

A System.Threading.Tasks.Task representing the asynchronous operation.

Remarks

This method performs the following steps: 

<ol><li>Determines the output directory for the published application.</li><li>Cleans up existing publish directories if specified.</li><li>
            Applies version transformations to project files if
            <xref href="DecSm.Atom.Module.Dotnet.Helpers.DotnetPublishAndStageOptions.SetVersionsFromProviders" data-throw-if-not-resolved="false"></xref> is <code>true</code>.
        </li><li>
            Executes the `dotnet publish` command using
            <xref href="DecSm.Atom.Module.Dotnet.Cli.IDotnetCli.Publish(DecSm.Atom.FileSystem.RootedPath%2cDecSm.Atom.Module.Dotnet.Cli.PublishOptions%2cDecSm.Atom.Process.ProcessRunOptions%2cSystem.Threading.CancellationToken)" data-throw-if-not-resolved="false"></xref>
            .
        </li><li>Moves the published output to the Atom publish directory for the project.</li></ol>

The BuildVersion is used for version transformation.

DotnetPublishAndStage(string, DotnetPublishAndStageOptions?, CancellationToken)

Publishes a .NET project by its name and stages the output.

[PublicAPI]
Task DotnetPublishAndStage(string projectName, DotnetPublishAndStageOptions? options = null, CancellationToken cancellationToken = default)

Parameters

projectName string

The name of the project to publish (e.g., "MyWebApp").

options DotnetPublishAndStageOptions

Optional. Configuration options for the publishing and staging process.

cancellationToken CancellationToken

A cancellation token to observe while waiting for the task to complete.

Returns

Task

A System.Threading.Tasks.Task representing the asynchronous operation.

Remarks

This method first locates the project file based on its name, then calls DotnetPublishAndStage(RootedPath, DotnetPublishAndStageOptions?, CancellationToken) with the resolved path.

Exceptions

StepFailedException

Thrown if the project file cannot be located.

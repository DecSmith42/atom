Defines methods for customizing the host configuration during application startup.

[PublicAPI]
public interface IConfigureHost

Remarks

This interface is automatically implemented on the build definition via source generation if any inherited interfaces use ConfigureHostAttribute or ConfigureHostBuilderAttribute. It provides hooks for modifying the host builder and the host itself. This interface is intended for internal framework use.
